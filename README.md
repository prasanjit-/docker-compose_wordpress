A demo on installing a Wordpress Blog Cluster with Docker Compose


Architecture: Sets up a cluster with webnodes (Apache+php+wordpress) & Mysql Database Node


# Commands-

To check process:
```
#docker-compose ps 
```
To start the cluster
```
#docker-compose up -d
```

To stop the cluster
```
#docker-compose down
#docker-compose down --volumes
```
